CICLO DE VIDA DEL MACHINE LEARNING

Machine Learning comprende de 5 pasos los cuales son:

01

Recopilación de datos

Recolección de un conjunto de datos.

02

Preparación de datos

Tratamiento de los datos y control de calidad.

03

Modelo de desarrollo

Construcción de un modelo optimo y eficiente.

04

Evaluación del modelo

Prueba la seguridad y fiabilidad del modelo.

05

Implementación del modelo

Despliegue del sistema con resultados precisos, rápidos y aceptables.
